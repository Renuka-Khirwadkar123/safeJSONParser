Safe JSON Parser 🔐



    

    

  
🧠 About This Project

  
  

    I built this project to understand TypeScript concepts like 
    Type Guards and Type Narrowing using a real-world example.
  


  

    This Safe JSON Parser validates user input instead of blindly trusting data, 
    ensuring better reliability and fewer runtime errors.
  


  

    It demonstrates how to safely handle unknown data using custom type guards.
